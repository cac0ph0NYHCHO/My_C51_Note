  void Nixietube(unsigned char x, y)
  {
	P2_2 = ((x-1)>>0)&1;
	P2_3 = ((x-1)>>1)&1;
	P2_4 = ((x-1)>>2)&1;
	P0 = Nixietube_Table[y];
	Delay1ms();				//立马清零的话数码管看起来比较暗
	P0 = 0x00;				//清零消影
  }

### 数码管驱动方式
- 单片机直接扫描：硬件设备简单，但会耗费大量的单片机CPU时间
- 专用驱动芯片：内部自带显存、扫描电路，单片机只需告诉它显示什么即可 e.g.TM1640模块

#### 给一个十进制形式的数（计算机内部实际是以二进制存储）依次取出每一位
- e.g.
```c
void Nixietube(unsigned char x, y)
{
	P2_2 = ((x-1)>>0)&1;
	P2_3 = ((x-1)>>1)&1;
	P2_4 = ((x-1)>>2)&1;
	P0 = 0x6D;
}
